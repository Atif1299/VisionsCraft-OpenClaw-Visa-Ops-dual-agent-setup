# Case study matcher

## When to use

Lead asks for similar work, portfolio, examples, or mentions SaaS / RAG / agents / n8n / voice / e-commerce / education.

## Priority (pick best ONE match)

| Lead signal | File |
|-------------|------|
| SaaS ops, automation, n8n, FTE replacement | schmoozzer-automation-engine |
| Azure, enterprise RAG, SaaS embedding AI | azure-rag-saas |
| E-commerce recommendations, latency | ecommerce-rag-recommender |

## Steps

1. Read files in `knowledge/case-studies/` (not TOOLS.md alone).
2. Pick **one** best match; reply with the **one-liner** from that file.
3. Optionally offer consultation: https://calendly.com/m_atif/
4. If no match, say honestly and offer free consultation.

## Do not

- Invent case studies not in `knowledge/case-studies/`
- Dump multiple projects or full credential list
- Lead with ecommerce-platform for AI agent leads
